# السراج للحلول الصناعية | Al-Seraj Industrial Solutions 🏢🔥

المستودع الرسمي والموحد لشركة **السراج للتجهيزات الصناعية**، الوكيل المعتمد لشركة **Asenware** العالمية في الجمهورية اليمنية. تم تطوير هذا المستودع ليكون منصة شاملة تضم جميع إصدارات وتحديثات الموقع الإلكتروني للشركة، مع التركيز على أحدث معايير الأمان الصناعي والحلول المتكاملة.

[معاينة الموقع المباشر 🌐](https://karar011.github.io/seraj-industrial-solutions/)

---

## 🚀 نظرة عامة على المستودع الموحد | Unified Repository Overview

تم دمج عدة مستودعات سابقة (مثل `seraj-website`, `seraj-asenware-yemen`, `saraj-website`, `asserag-website`) في هذا المستودع الواحد لتبسيط إدارة المشاريع وتوحيد جهود التطوير. يحتوي هذا المستودع على أحدث إصدار من الموقع الإلكتروني للشركة، بالإضافة إلى أرشيف للإصدارات السابقة في مجلد `versions`.

## ✨ الميزات الرئيسية | Key Features

### 1. **تصميم حديث ومستجيب**
- واجهة مستخدم جميلة وسهلة الاستخدام.
- دعم كامل للأجهزة المحمولة والشاشات المختلفة.
- تأثيرات حركية سلسة ومحترفة.

### 2. **دعم اللغات**
- دعم كامل للغة العربية والإنجليزية.
- تبديل فوري بين اللغات دون إعادة تحميل الصفحة.
- اتجاهات نصية صحيحة (RTL/LTR).

### 3. **الأدوات التفاعلية**
- **التحقق من الأصالة**: أداة للتحقق من أرقام المنتجات الأصلية.
- **حاسبة الأنظمة**: أداة لتقدير احتياجات منشأتك من الأنظمة بناءً على المساحة وعدد الغرف.

### 4. **أقسام الموقع**
- **الرئيسية**: عرض تقديمي للشركة والخدمات.
- **خدماتنا**: تفاصيل الخدمات المقدمة (أنظمة الحريق والإنذار المبكر، عقود الصيانة الدورية، التدريب الفني والاستشارات).
- **الأدوات**: أدوات تفاعلية للعملاء.
- **التحميلات**: مركز تحميل الكتالوجات والشهادات الدولية.
- **اتصل بنا**: معلومات التواصل والموقع.

### 5. **الميزات الإضافية**
- زر WhatsApp عائم للتواصل المباشر.
- شريط إعلانات للعروض الخاصة.
- نظام الوضع الليلي والنهاري.
- تحميل سريع مع شاشة تحميل احترافية.
- تحسين محركات البحث (SEO).

## 🚀 كيفية الاستخدام | How to Use

### التشغيل المحلي
1. انسخ الملفات إلى مجلد محلي.
2. افتح ملف `index.html` في متصفح الويب.
3. أو استخدم خادم محلي:
```bash
# باستخدام Python 3
python -m http.server 8000

# ثم افتح http://localhost:8000
```

### النشر على GitHub Pages
1. ادفع الملفات إلى مستودع GitHub.
2. اذهب إلى إعدادات المستودع.
3. فعّل GitHub Pages من فرع `main`.
4. سيكون الموقع متاحاً على `https://karar011.github.io/seraj-industrial-solutions`
